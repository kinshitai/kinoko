## Quick Install (from source)

Kinoko is currently installed from source. Pre-built binaries are not yet available.

```bash
git clone https://github.com/kinshitai/kinoko.git
cd kinoko
go install ./cmd/kinoko
```

Verify:
```bash
kinoko --version
```

## Requirements

### Go 1.24+

```bash
go version
# Should output: go version go1.24.x or later
```

Install Go: https://golang.org/dl/ or via package manager:
- **macOS:** `brew install go`
- **Ubuntu/Debian:** `sudo apt install golang-go` (check version — distro packages may be old)
- **Windows:** Official installer or `scoop install go`

### Git

```bash
git --version
```

Usually pre-installed. If not:
- **macOS:** `brew install git` or Xcode Command Line Tools
- **Ubuntu/Debian:** `sudo apt install git`
- **Windows:** https://git-scm.com/download/win

### SSH Client

```bash
ssh -V
```

Pre-installed on macOS, most Linux distros, and Windows 10+.

## Post-Install Setup

Kinoko uses three commands — one for setup, one for infrastructure, one for the local daemon:

```bash
# 1. Initialize workspace (config, SSH key, cache dir)
kinoko init

# 2. Start shared infrastructure (git server, API, hooks)
kinoko serve

# 3. Start local agent daemon (workers, scheduler, injection)
kinoko run
```

For team setups, connect to an existing server instead:

```bash
kinoko init --connect your-server:23231
kinoko run
```

See the [Quickstart](/quickstart/) for a complete walkthrough and [Architecture](/concepts/architecture/) for deployment patterns.

## Common Issues

### "command not found: kinoko"

Your Go bin directory isn't in PATH:

```bash
echo 'export PATH=$PATH:$(go env GOPATH)/bin' >> ~/.bashrc
source ~/.bashrc
```

Adjust for your shell (`~/.zshrc` for Zsh, `~/.config/fish/config.fish` for Fish).

### "go: command not found"

Go isn't installed or isn't in PATH. Install from https://golang.org/dl/ and ensure `/usr/local/go/bin` is in PATH.

## Uninstall

```bash
rm $(which kinoko)
rm -rf ~/.kinoko  # WARNING: deletes all local skills, config, and cache
```
