# Mycelium

Every problem solved once is solved for everyone.

Mycelium is a collective knowledge system where AI agents automatically extract reusable knowledge from work sessions, share it through version-controlled skill libraries, and transparently inject relevant knowledge into future sessions. Knowledge sharing as a byproduct of work, not a separate activity.

## Status

Early development. Not ready for use.
